# 🐍 Python Projects | Proyectos Python

Personal projects built while learning Python. Each one integrates core concepts like data structures, file handling, error management, and data analysis with pandas.

Proyectos personales desarrollados durante el aprendizaje de Python. Cada uno integra conceptos clave como estructuras de datos, manejo de archivos, gestión de errores y análisis de datos con pandas.

---

## 📊 Expense Analyzer | Analizador de Gastos
**`/expense-analyzer`**

Reads a CSV file with personal expenses, groups them by category, identifies the highest spending category, finds the most expensive individual purchase, calculates the average spend, and exports a summary report.

Lee un archivo CSV con gastos personales, agrupa por categoría, identifica dónde se gasta más, encuentra el gasto individual más alto, calcula el promedio y exporta un reporte resumen.

**Stack:** Python, pandas, CSV

---

## 📒 Contact Manager | Gestor de Contactos
**`/contact-manager`**

CLI contact book with full CRUD — add, search, delete, and list contacts. Validates phone numbers and email format. Persists data to a .txt file and loads it automatically on startup. Built with a modular architecture separating UI from business logic.

Agenda de contactos por línea de comandos con CRUD completo — agregar, buscar, eliminar y listar. Valida formato de teléfono y email. Persiste datos en .txt y los carga automáticamente al iniciar. Arquitectura modular separando interfaz de lógica de negocio.

**Stack:** Python, archivos .txt, módulos

---

## 📝 Text Analyzer | Analizador de Texto
**`/text-analyzer`**

Reads any .txt file and produces a full linguistic analysis — total word count, unique words, top 5 most frequent words (excluding stopwords), and the longest word. Exports the report to a new .txt file.

Lee cualquier archivo .txt y produce un análisis lingüístico completo — total de palabras, palabras únicas, las 5 más repetidas (excluyendo palabras vacías) y la más larga. Exporta el reporte a un nuevo archivo .txt.

**Stack:** Python, archivos .txt, sets, diccionarios
